Установка Git

Щоб користуватись Git, необхідно його встановити. Git — безкоштовна програма з відкритим початковим кодом, яку можна завантажити на сайті https://git-scm.com/download/.

Linux

Користувачам операційної системи Linux зручно встановити Git за допомогою пакетних менеджерів, вбудованих у дистрибутив. Наприклад, для дистрибутивів, заснованих на Debian варто спробувати таку команду (Примітка: тут і далі $ перед командами означає виконання від імені користувача і не є текстом самої команди, тобто не вводиться):

bash

sudo apt install git-all

$

Для Fedora та RPM-подібних дистрибутивів:

bash

sudo dnf install git-all

$

macOS

В операційній системі macOS система контролю версій Git надається у комплекті Xcode Command Line Tools, які зазвичай уже встановлені у користувачів, що займаються програмуванням. Якщо цей інструментарій не встановлений, для початку його установки досить виконати команду і дотримуватись інструкцій на екрані:

bash

git --version

$

Також Git можна встановити на macOS за допомогою інсталятора, який надається на сайті https://git-scm.com/download/mac/ або за допомогою пакетного менеджера Homebrew:

bash

brew install git

$

Windows

Користувачам операційної системи Windows зручніше за все буде скачати та встановити Git за допомогою інсталятора, наданого на сайті https://git-scm.com/download/win/.

Перевірка установки

Після завершення установки Git в операційній системі користувача варто перевірити працездатність Git за допомогою команди:

bash

git --version

$

Якщо вивід приблизно такий, Git встановлений і готовий до роботи:

bash

git --version

git version 2.19.1

$
