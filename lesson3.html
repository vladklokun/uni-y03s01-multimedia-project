Первинне налаштування Git

Перед використанням Git його необхідно налаштувати. Для налаштування Git комплектується утилітою git config. В ієрархії конфігурацій є 3 рівні у порядку спадання пріорітету:

  1. Рівень проектної конфігурації — містить налаштування поточного проекту, які зберігаються у поточній директорії Git (файл .git/config);
  2. Рівень користувацької конфігурації — містить налаштування, створені поточним користувачем Git, які зберігаються у домашній директорії користувача (файл ~/.gitconfig у Unix-подібних системах або $HOME\.gitconfig для Windows);
  3. Рівень системної конфігурації — містить налаштування, які розповсюджуються на всю систему. Вони зберігаються у системних директоріях (файл /etc/gitconfig у Unix-подібних системах або C:\ProgramData\Git\config у Windows Vista та пізніших).

Ваша особа

Git використовує інформацію про особу — ім'я та адресу електронної пошти — у кожному коміті, тому при первинному налаштуванні в першу чергу необхідно додати саме її. Можна встановити дані про особу глобально, щоб вони використовувались для всіх проектів, за допомогою такої команди:

bash

git config --global user.name "Thomas Anderson"

$ git config --global user.email "tomanderson@example.com"

$

Щоб використати інші дані для певного проекту, досить прибрати опцію --global.

Текстовий редактор

Часто при роботі з Git виникає необхідність вводу додаткового тексту зі сторони користувача. Щоб зробити цю процедуру зручною, Git пропонує обрати та встановити за замовчуванням улюблений текстовий редактор користувача. Наприклад, щоб змусити Git використовувати Neovim в якості текстового редактора за замовчуванням, необхідно виконати команду:

bash

git config --global core.editor nvim

$

Популярним графічним текстовим редактором для операційної системи Windows є Notepad++. Якщо він встановлений у директорії C:\Program Files (x86)\Notepad++\, то для нього команда виглядатиме так (для позначення директорій використовується передній слеш):

bash

git config --global core.editor "'C:/Program Files (x86)/Notepad++/notepad++.exe' -multiInst -nosession"

$

Перевірка налаштувань

Щоб перевірити значення існуючих налаштувань, досить ввести команду:

bash

git config --list

$
