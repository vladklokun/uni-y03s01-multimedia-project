Документація

Git — надзвичайно гнучка система з величезною кількістю функцій і можливістю детального налаштування, тому для покращення розуміння цих функцій користувачу надається документація. Цю функцію виконує утиліта git-help, яка надає детальну інформацію з використання будь-якої команди, вбудованої в Git. Вона використовується так:

bash

git help <інша утиліта>

$

Наприклад, щоб отримати детальну інформацію про утиліту git-add, необхідно ввести команду:

bash

git help add

$

Щоб отримати стислу інформацію про команду, треба запустити цікавлячу команду з параметром -h або --help. Наприклад, для git-add:

bash

git add -h

$
